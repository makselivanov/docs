Где:
* `--panic-threshold` — порог для режима паники.
* `--http-healthcheck` — параметры проверки состояния ресурсов:
  * `port` — порт.
  * `healthy-threshold` — порог работоспособности.
  * `unhealthy-threshold` — порог неработоспособности.
  * `timeout` — таймаут.
  * `interval` — интервал.
  * `host` — адрес хоста.
  * `path` — путь.
